VS Code + Github Pages + Hexo 写博客

Posted on 2021-01-11 Disqus: 
Update @ 20210111

hexo安装

注册Github

建立一个Github repository放博客
reponsitory的名字要是dustycn.github.io(待确认是否只能使用用户名）

安装VS Code及插件

安装Node.js

安装Git

(参考：https://juejin.cn/post/6844903651341762573）

打开git bash设置SSH远程连接：

生成密钥：

如果不需要密码，连续三个回车，得到两个文件id_rsa和id_rsa.pub，位置：
C:\Users<用户名>.ssh
打开复制 id_rsa.pub内容到github上。

安装Hexo

新建一个BLOG文件夹并再VS Code中打开;
在VS Code的终端运行：

编辑配置文件_config.yml：

deploy:
type: git
repo: #这里填入你之前在GitHub上创建仓库的完整路径，记得加上 .git Example: <https://github.com/dustycn/dustycn.github.io.git>
branch: master #2020年10月1日之后推荐使用main取代master,避免歧视。

安装部署Github插件

绑定域名

域名解析里面设置一个A记录（待确认是否一定要）和一个CNAME
A记录的的IP值可以ping dustycn.github.io获得
进入博客repository,设置custom domain
进入本地blog/source文件夹下，新建一个CNAME不带扩展名的文件，填入custom domain，保存即可。注意 custom domain不要带www,否则访问的时候不带www无法访问。

安装主题

cd进入blog文件夹下:

注意，最先的版本是theme-next，但是原作者好久不更新了，next-theme的作者叶是大神，更新比较频繁，推荐。
（参考：http://zuyunfei.com/2020/06/12/the-next-theme/）

设置hexo配置文件_config.yml：

theme:next

发布(n g d “三联”）

  1. hexo clean （如果显示正常不需要）
  2. hexo n(new) “blog article title”
  3. hexo g (generate)
  4. hexo s (server)
  5. hexo d(deploy)

发布图片

  1. 使用markdown paste插件插入的本地图片只能在首页显示；
  2. 有其它方式可以在文章中显示，是在每个发布的文章的文件夹中，建立相同文件名的文件夹存放该文章所用到的图片（待尝试）；
  3. 目前推荐使用picgo插件，直接上传到github，粘贴链接

添加评论系统

我使用的是disqus.com，注册后取得shortname后，变更hexo配置文件_config.yml：

disqus:
enable: true
shortname: dustycnblog
count: true

添加Github猫

（参考：https://sinyilin.github.io/hexo/20191109/1819073706/）
去https://tholman.com/github-corners/#选择样式，

变更为你自己的地址：

<div class="headband"></div> 下一行即可。
